# Photographer Portfolio Website 📸

אתר פורטפוליו מקצועי ומעוצב עבור צלמת, המציג תיק עבודות מגוון, המלצות לקוחות וטופס ליצירת קשר פעיל.

## 🚀 Live Demo
[צפו באתר החי ב-GitHub Pages](https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/)

## ✨ תכונות עיקריות (Features)
- **תיק עבודות מחולק לקטגוריות:** ניו-בורן, צילומי סטודיו, חוץ, אירועים, נופים ועוד.
- **טופס יצירת קשר פעיל:** אינטגרציה מלאה עם Formspree לקבלת פניות ישירות למייל.
- **חווית משתמש (UX):** ניווט מהיר בין דפי השירותים, ההמלצות ותיק העבודות.
- **עיצוב רספונסיבי:** התאמה מלאה למכשירים ניידים וטאבלטים.
- **עיצוב נקי ויוקרתי:** שימוש בפלטת צבעים רכה המתאימה לעולם הצילום.

## 🛠 טכנולוגיות (Tech Stack)
- **HTML5:** מבנה סמנטי של דפי האתר.
- **CSS3:** עיצוב מתקדם, Flexbox ו-Grid, אנימציות מעבר חלקות.
- **Formspree:** ניהול צד שרת עבור טופס יצירת הקשר.
